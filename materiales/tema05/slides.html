La Gramática de los Gráficos

Idea: Construir gráficos por capas

  • Datos: ¿Qué queremos graficar?
  • Estéticas (aes): ¿Qué variables van en x, y, color, etc.?
  • Geometrías (geom): ¿Puntos, líneas, barras?

Anatomía de un Gráfico ggplot2

ggplot(data = datos, aes(x = variable_x, y = variable_y)) +
  geom_point()
  1. ggplot(): inicializa el gráfico
  2. aes(): mapea variables a escalas
  3. geom_*(): especifica tipo de gráfico

Definir una geometría

El operador + encadena capas

El Primer Gráfico: 5 de 5

Añadir atributos visuales

1ggplot(iris, aes(x = Sepal.Length, y = Sepal.Width, color = Species)) +
2  geom_point(size = 3)
1
color = Species mapea la especie a colores diferentes
2
size = 3 aumenta el tamaño de los puntos

Estéticas: Más Allá del Color

Mapea variables a atributos visuales:

ggplot(
  iris,
  aes(
    x = Sepal.Length,
    y = Sepal.Width,
1    color = Species,
2    size = Petal.Length,
3    shape = Species,
4    alpha = Sepal.Length
  )
) +
  geom_point()
1
color - cambia el color de los puntos
2
size - cambia el tamaño según una variable
3
shape - cambia la forma (requiere variable categórica)
4
alpha - cambiar transparencia (0 = transparente, 1 = opaco)

Gráfico de Barras

Conteo simple:

1ggplot(iris, aes(x = Species)) +
2  geom_bar()
1
Especificar solo x, sin y
2
geom_bar() cuenta automáticamente las observaciones

Con valores específicos:

resumen <- iris |>
  group_by(Species) |>
  summarise(promedio = mean(Sepal.Length))

1ggplot(resumen, aes(x = Species, y = promedio)) +
2  geom_col()
1
Usar datos pre-resumidos
2
geom_col() para valores específicos

Histogramas

ggplot(iris, aes(x = Sepal.Length)) +
  geom_histogram(bins = 20)

Tip

bins controla el número de barras

Diagramas de Caja (Boxplot)

ggplot(iris, aes(x = Species, y = Sepal.Length)) +
  geom_boxplot()

Muestra: mediana, cuartiles, valores atípicos

Gráfico de Líneas

Gráfico de Violín

ggplot(iris, aes(x = Species, y = Sepal.Length, fill = Species)) +

1

Gráfico de Densidad

ggplot(iris, aes(x = Sepal.Length, fill = Species)) +
1  geom_density(alpha = 0.5)
1
alpha = 0.5 añade transparencia para ver distribuciones superpuestas

Personalizando Títulos y Etiquetas

ggplot(iris, aes(x = Sepal.Length, y = Sepal.Width, color = Species)) +
  geom_point() +
  labs(
1
    title = "Características de las Flores",
    subtitle = "Dataset Iris",
    x = "Largo del Sépalo (cm)",
    y = "Ancho del Sépalo (cm)",
    color = "Especie"
  ) +
2  theme_minimal() +
3  theme(plot.title = element_text(size = 14, face = "bold"))
1
labs() personaliza títulos y etiquetas
2
theme_minimal() aplica un tema predefinido
3
theme() para ajustes finos del estilo

Bonus: Esquisse para Gráficos Interactivos

¿Quieres explorar gráficos sin escribir código? Prueba esquisse:

# Instalar (una sola vez)
install.packages("esquisse")

# Usar esquisse
library(esquisse)
esquisser(iris) # Abre interfaz gráfica interactiva

Tip: Usa esquisse para exploración rápida, luego copia el código final en tu script
